2. Edit the Default Routing

Configure the default routing after the user registers or logs in. We will use an index.blade.php instead of the home.blade.php.

  1. Delete welcome.blade.php and home.blade.php. Both views will not be used.

    PATH: backend>resources>views

  1. Create a new folder: posts
    Create index.blade.php inside. This will serve as the homepage.

    PATH: backend>resources>views>posts>index.blade.php

    EXPECTED OUTPUT:

  1. Add the basic directives in index.blade.php

    2. PATH: backend>resources>views>posts>index.blade.php

                    @extends('layouts.app')
                @section('title', 'Home')
                @section('content')
                <div style="margin-top: 100px">
                    <h2 class="text-muted text-center">No posts yet.</h2>
                    <p class="text-center">
                        <a href="#" class="text-decoration-none">Create a new post</a>
                    </p>
                </div>
                @endsection
  1. Edit the index() method of HomeController. This is temporary. We will use PostController later for the index.

    2. PATH: backend>app>Http>Controllers>HomeController.php

                    public function index(){
                    return view('posts.index');
                }
