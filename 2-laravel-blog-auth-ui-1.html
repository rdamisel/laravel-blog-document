Auth with UI

1. Install Auth with UI

  • $ npm -v
    • Do this command to check if Node has already been installed.
    • If not, go to https://nodejs.org/en/download/ . Then, download the installer.
    1. $ docker-compose exec app composer require laravel/ui
      • Install laravel/ui Composer package
    1. $ docker-compose exec app php artisan ui bootstrap --auth
      • Generate Auth scaffolding
      • Install Bootstrap scaffolding
    1. $ cd backend
      • The next steps shall be done inside the backend folder because this is the location of Laravel.
    1. $ npm install && npm run dev
      • Install npm dependencies
      • Download Bootstrap packages
    1. $ Ctrl / control + C
      • Exit to be able to execute the next command
    1. $ npm run build
      • Build asset files in public/build/assets

Once you are done running the commands above, go to web-1 or localhost:82 and click Login or Register at the upper right-hand corner.
